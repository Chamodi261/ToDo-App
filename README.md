# 📝 ToDo App

A simple and responsive React ToDo application built with 💙. It allows you to:

✅ Add tasks  
🗑️ Delete tasks  
📖 Read and view tasks  
✏️ Update tasks  

---

## 🚀 Features

- ✨ Clean and modern UI
- ➕ Add new todos
- 📝 Update existing tasks
- 🗑️ Remove tasks
- 🔄 Real-time list updates
